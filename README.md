ソフトウェア概要
--------------------------------------------------------------------------------

本ソフトウェアは、特定のフォルダ内のpngファイルを一括で左右反転し、必要ならファ
イル名の一部を変更して保存する作業を簡略化するためのツールです。

主にイラスト制作者の素材配布時の定型的作業の負担軽減を目的としています。

ファイル構成
--------------------------------------------------------------------------------

zip配布時の構成

- README.html          : READMEへのリンク
- flip.bat             : **実行ファイル**
- flip_image.jar       : プログラム本体
- flip_interactive.bat : **対話式実行ファイル**
- java_download.html   : Javaアップデート公式サイトへのリンク

### flip.batとflip_interactive.bat の違い

#### flip.bat

flip.batはダブルクリックだけで自動的にすべての対象フォルダ内のpngファイルを左右
反転することを目的とした実行ファイルです。

flip.batは初期状態では自動的に画像を加工できません。

ワンアクションで自動的に処理をさせるために、最初の一度だけユーザが設定を記述する
必要があります。

最初の一度と、その時設定したフォルダ名とファイルの命名規則に従う限り、ワンアク
ションですべての処理を完了することができます。

ただし設定の仕方を誤った場合に正常にプログラムが実行できなくなるリスクがありま
す。

#### flip_interactive.bat

flip_interactive.batは対話式プログラム実行ファイルです。  
すべての設定は対話式のキー入力によって行います。

flip.batと違い、flip_interactive.batは初回設定を必要とせず、ダブルクリックで常に
動作します。

ただし、実行するたびに対象となるフォルダと保存先フォルダ、置換するタグといったオ
プションをユーザ自身が入力する必要があります。

これはユーザ独自の命名規則を持たない、その都度異なる形式のファイル名を付けてしま
うユーザにとって有力となるでしょう。

動作条件
--------------------------------------------------------------------------------

本ソフトを実行するにはJavaがインストールされている必要があります。また、本ソフト
作成時のJavaのバージョン以上がインストーされている必要があります。

もし本ソフトをダブルクリックで実行できなかった場合は、Webブラウザで下記URLの公式
サイトを開くか、同封の java_download.html をダブルクリックして公式サイトを開いて
Javaをインストールまたはアップデートしてください。

[Javaダウンロード](https://java.com)

それでも起動できなかった場合は、お手数ですがFAQの項目を確認した後に、本ドキュメ
ント先頭の連絡先より作者にご報告ください。

実行方法
--------------------------------------------------------------------------------

"flip.bat"または"flip_interactive.bat"をダブルクリックしてください。  
LinuxOSをご使用の方はターミナルから"java -jar mvte.jar"と打ち込んでください。

使い方
--------------------------------------------------------------------------------

### flip.batの使い方

初回起動時にWindows環境であればおそらくメモ帳が起動するでしょう。

開いたファイルは選択したflip.batです。

メモ帳で(必ずしもメモ帳である必要はありませんが)ファイルの先頭付近にある

```bat
set img_from=  
set img_to=  
set from_tag=  
set to_tag=  
```

という部分の = 以降にテキストを追記します。

この時、=の前後に**空白を含めない***ようにしてください。  
正常に動作しなくなる可能性があるからです。

それぞれの設定項目は次のような意味を持ちます。

- img_from  
  左右反転画像の元画像のみを格納しているフォルダ名

- img_to  
  左右反転した後の画像を保存するフォルダ名  
  (存在しないフォルダ名の場合は自動で生成します)

- img_from  
  左右反転画像の元画像のみを格納しているフォルダ名

- img_from  
  左右反転画像の元画像のみを格納しているフォルダ名

入力が完了したらflip.batを上書き保存してダブルクリックしてください。

コンソール画面が表示され、正常に保存が完了できたことが画面に表示されたら、設定は
うまく言っているということになります。

もしこの時、画像の保存に成功したと読み取れない文章が画面に表示された場合、設定
ファイルの値や、flip_image.jarがflip.batと同じフォルダ内に存在するか確認してくだ
さい。

### flip_interactive.batの使い方

ダブルクリック直後にコンソールが表示されます。

指示に従ってテキストを入力していただければ正常に動作するはずです。

FAQ
--------------------------------------------------------------------------------

### Q1. 実行ファイルをダブルクリックしても動作しない

ご利用の環境にJavaがインストールされているか確認してください。また、Javaがインス
トールされていても、本ソフト作成時のJavaのバージョン以下をご利用の場合、動作しな
い場合があります。

Javaのインストール、アップデート方法については[動作条件](#動作条件)の項目を参照
してください。

### Q2. READMEのレイアウトが崩れている

仕様です。フォントが異なるとレイアウトが崩れて見えることがあります。

メモ帳などでこのドキュメントを開いているのでしたら、書式メニューのフォントからMS
ゴシック(MS Pゴシックではありません)を選択すると作者が期待している通りのレイアウ
ト表示になります。
